Mon CV interactif en ligne est une application web moderne qui présente mon parcours professionnel, mes compétences techniques et mes projets de façon dynamique et responsive.

## Caractéristiques principales

- **Design moderne** : Interface élégante et professionnelle avec animations subtiles
- **Sections interactives** : Navigation fluide entre les différentes sections du CV
- **Responsive design** : Expérience optimisée sur tous les appareils
- **Mode sombre/clair** : Option de personnalisation de l'affichage
- **Téléchargement PDF** : Possibilité de télécharger une version PDF du CV

Développé avec Next.js et TypeScript, ce projet utilise les composants shadcn/ui et Tailwind CSS pour une interface utilisateur soignée et cohérente, tout en maintenant d'excellentes performances et une accessibilité optimale.
