FinanceAI est une application web complète pour la gestion des finances personnelles, offrant un tableau de bord intuitif pour suivre vos dépenses, revenus et objectifs financiers.

## Fonctionnalités principales

- **Gestion multi-comptes** : Suivi de plusieurs comptes bancaires et portefeuilles
- **Transactions catégorisées** : Organisation des dépenses et revenus avec système de catégories personnalisables
- **Budgets intelligents** : Création et suivi de budgets mensuels par catégorie
- **Rapports visuels** : Visualisation des tendances financières avec graphiques interactifs
- **Authentification sécurisée** : Connexion via Google, Discord ou email/mot de passe
- **Expérience utilisateur intuitive** : Interface moderne avec thème clair/sombre

## Stack technique

Cette application utilise une architecture moderne basée sur Next.js avec le App Router, combinant TypeScript pour la sécurité du typage, tRPC pour des API typées, et Drizzle ORM pour l'interaction avec la base de données PostgreSQL hébergée sur Supabase.
